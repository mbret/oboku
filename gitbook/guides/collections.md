# Collections

## Collections vs Series

By default a shelve is a collection, a simple folder containing items. In order to start having more metadata and features enabled, you can turn a collection into a series. By doing so, oboku will try to fetch metadata for existing matching series (eg: published year, publisher, number of books, etc). Series may also offer a better reading experience by suggesting continuation etc.

## Convert a collection into a series

To make a collection a series you can either mark it in the app or use a [directives.md](directives.md "mention").

## Series metadata

Metadata are collected for series only and provide extra information when possible. We try to gather information such as rating, if the series is still ongoing, start year etc.

### Metadata Refresh

Metadata for a series are automatically refreshed on two occasion:

* **sync**: Your collection will be refreshed if it has not been refreshed yet or if the link has been updated recently
* **manual**: You can force a metadata refresh manually on the collection detail page.
