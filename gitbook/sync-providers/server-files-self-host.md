# Server Files (self host)

Whenever server sync is enabled on an oboku instance you will be able to synchronize your content directly through the `server` provider. Visit the [server-sync.md](../self-hosting/server-sync.md "mention") section for its configuration.

## Sync behavior

When you sync you have no selection options so all of the configured sources will be synchronized. oboku will not consider the source folder as a collection itself and only start iterate on the sub items.

This way you are free to rename or move your path wherever you want without affecting users library.&#x20;
