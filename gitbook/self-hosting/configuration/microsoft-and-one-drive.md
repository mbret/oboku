# Microsoft & One Drive

## Setup Microsoft account

* Create a developer account on [https://portal.azure.com](https://portal.azure.com/auth/login/)
* Add a new application
* Enable the permissions required by the [https://learn.microsoft.com/en-us/onedrive/developer/controls/file-pickers/?view=odsp-graph-online](https://learn.microsoft.com/en-us/onedrive/developer/controls/file-pickers/?view=odsp-graph-online)
* Add a single page authentication and configure the redirect URL as [https://app.oboku.me/auth-callback-microsoft](https://app.oboku.me/auth-callback-microsoft) but replaces the origin with your own web url.

## Admin Panel

Once your account is setup, you can provide the required information in the admin panel
