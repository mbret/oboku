# Installation & usage

oboku app being a website first you can and should use it by visiting [https://app.oboku.me](https://app.oboku.me). Once you visit the website you have then two choices, you can either stay and use it with your favorite internet browser, or you can **install it to your home screen**.&#x20;

{% hint style="info" %}
Remember that oboku is an **offline first app**. It means that after opening it for the first time, you can open it and use it whether you have internet connection or not. All your changes will be sync later with your other devices. This really is not much different from a native app.
{% endhint %}

By installing the app to your home screen you will be able to open it the same way as you would open a native app (by taping icon) and take benefit of the **fullscreen mode**.

Depending of your browser, device brand and version **the option to install it to home screen might not be available**. In this case you can use the native app version (see below).

**For more information about device compatibilities please visit the dedicated wiki section.**

## Native App

{% hint style="warning" %}
Native app are not yet available on the store, if you want to try them out you can access it at this [shared drive](https://drive.google.com/drive/folders/1yPpzwMW6PFw3g9P_ed82M4jzvQZxhioG?usp=sharing). If you have issue with it please contact us.
{% endhint %}

Using oboku should be as conveniant as using a native app and in most case scenario you should be abe to install the web app to your homescreen. However it is not always possible on every devices such as old phones, old internet browser, or **eBook readers.** For theses devices you can use the App Store version which is in fact a native app wrapping the web app. The experience should remain the same.

## **Why don't you have official native apps?**

Deelopment is costly, the web is cool and works mostly everywhere. So far there are no technical limitation known by making it web based. The only problem is for devices that have either broken, outdated or no web browser at all (remarkable, kobo, etc). Developing a native app for these devices would require an entire new development and we just don't have the resources yet. If you feel like helping doing it, your help would be much appreciated.
